# nx-date-time
> Date time utility for nx.

## installation
```rb
# from gem
gem 'nx-date-time'
# from git
gem 'nx-date-time', git: 'git@github.com:afeiship/nx-date-time.git'
```

## usage
```rb
Nx::DateTime::hello

# hello world
```

## build/publish
```shell
# build
gem build nx-date-time.gemspec

# publish
gem push nx-date-time-0.1.0.gem
```
